THE BRIEF

I WANTED TO DESIGN A LOGO THAT DESCRIBED MAINLY 3 IDEAS THAT I LIVE BY WHILE CREATING ART:
✦ CREATIVITY
✦ SIMPLICITY
✦ FLEXIBILITY

GREEN IS MY FAVOURITE COLOR, SO I HAD TO DESIGN A COLOR PALETTE THAT FOCUSED ON GREEN.

THE DESIGN

THE LOGO INCLUDES 3 KEY IDEAS:
✦ THE INITIALS OF MY NAME KASHYAP RAYAS
✦ THE K AND R BEING SYMMETRIC BY FLIPPING K VERTICALLY TO PRODUCE R
✦ A COLOR PALETTE WITH GREEN AS THE HERO

THE ELEMENTS

I DESIGNED THE WEBSITE PROTOTYPE USING FIGMA AND PHOTOSHOP. LATER CODED THE WEBSITE USING HTML, CSS AND JAVASCRIPT. 6000 LINES OF CODE LATER, HERE IT IS!

THE WEBSITE IS A MEDIUM TO SHOWCASE THE CULMINATION OF MY WORK SO FAR, MY KNOWLEDGE GAINED AND MY LOVE FOR GRAPHIC DESIGN.
